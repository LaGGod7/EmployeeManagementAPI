#Employee Management API

A RESTful Employee Management API built using Spring Boot.

-> Features

- Add Employee
- Get All Employees
- Get Employee by ID
- Update Employee
- Delete Employee
- Search Employee by Name

-> Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker
- Maven

-> Future Improvements

- Validation
- Global Exception Handling
- DTOs
- JWT Authentication
